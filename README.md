# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 기능
 - 객체 생성 시 1 부터 9 까지 서로 다른 수로 이루어진 3자리의 수를 정하는 모델 생성
 - 유저가 입력한 값이 유효한 값인지 확인 후 원하는 형태로 변경하는 모델 생성
 - 랜덤으로 정해진 수와 유저가 입력한 수를 비교 확인하는 기능 <br/>
   (같은 수&&같은 자리에 있으면 스트라이크, 숫자만 같으면 볼, 같은 수가 없으면 낫싱)
 - 정답 입력시 재시작을 묻고 게임의 재시작 유무를 정하는 기능

## 예외사항
 - 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시킨 후 애플리케이션 종료
